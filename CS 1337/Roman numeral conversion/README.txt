This program will take an input file in the format presented and convert all Roman numerals to Arabic numbers,
And all Arabic numbers to Roman numerals. It also stores all the numbers in a linked list, there are options 
within the program to sort the linked list by Roman numeral or by Arabic numbers without swapping data between each node.
There is also a find function that will get a Roman numeral or an Arabic number and say "Found" if it is present in the linked list.
After the user exits the program it will output all the numbers both Roman and Arabic to a file named "numbers.txt" in the order they are in the linked list.
Demonstrations of these functions can be found in the demo.txt file.
