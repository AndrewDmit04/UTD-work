This program will take a input file in the format presented and convert all roman numerals to Arabic numbers,
And all Arabic numbers to roman numerals. It also stores all the numbers in a linked list, there are options 
within the program to sort the linked list by Roman numeral or by Arabic numbers without swapping data between each node.
There is also a find function that will get an Roman numeral or a Arabic number and say "Found" if it is present in the linked list.
Demostrations of these function can be found in the demo.txt file.
